# 法人電気料金 請求書チェックシート（A4 1枚）

対象: 高圧・特別高圧・低圧の法人需要家  
利用方法: 当月請求書を手元に置き、該当項目を1つずつ照合してください。

## 1. 基本情報
- [ ] 供給地点特定番号
- [ ] 検針期間・検針日
- [ ] 契約種別（高圧 / 特別高圧 / 低圧電力 / 低圧電灯）
- [ ] 契約電力（kW）
- [ ] 料金プラン名称・改定日

## 2. 基本料金
- [ ] 単価（円/kW）
- [ ] 力率割引・割増
- [ ] 契約電力の決定方式（協議・負荷率・最大需要）

## 3. 電力量料金
- [ ] 使用電力量（kWh）
- [ ] 時間帯別（夏季/その他季、昼間/夜間）
- [ ] 単価（円/kWh）

## 4. 燃料費調整額
- [ ] 単価（円/kWh、プラス/マイナスの符号）
- [ ] 上限設定の有無
- [ ] 前月との差分

## 5. 再エネ賦課金
- [ ] 単価（円/kWh）
- [ ] 減免制度の適用状況

## 6. その他の費目
- [ ] 市場価格調整額（該当する場合）
- [ ] 容量拠出金（該当する場合）
- [ ] 託送料金相当分
- [ ] 口座振替割引・電子請求割引

## 7. 合計確認
- [ ] 小計・消費税・端数処理
- [ ] 前月比・前年同月比
- [ ] 使用量変動要因の記録

## メモ
- 異常値を検知した項目は「見直しポイントを知る（/articles/review-points）」の Step 2: 比較準備 を参照してください。
- 社内共有用のテンプレートが必要な場合は /articles/internal-explanation の稟議書テンプレをご覧ください。
